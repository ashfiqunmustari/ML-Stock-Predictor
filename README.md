**Stock Predictor using LSTM, Django, and React**

This project predicts stock prices using a deep learning LSTM model and serves predictions through a web portal. The backend is built with Django REST framework, while the frontend uses React for an interactive user interface.

**Features**

Predict next-day stock closing prices using historical data.

LSTM-based deep learning model trained on Yahoo Finance data.

View evaluation metrics: MSE, RMSE, RÂ².

Visualizations: actual vs predicted prices, 100-day and 200-day moving averages.

Secure JWT-based authentication.

Responsive React frontend for entering stock tickers and viewing predictions.

**Technology Stack**

Backend: Django, Django REST Framework, TensorFlow/Keras

Frontend: React, Bootstrap

Database: SQLite (for authentication and user management)

Data: Yahoo Finance API

Authentication: JWT (JSON Web Tokens)
